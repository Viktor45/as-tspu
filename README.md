## Purpose of the list
This list aims to simplify for end users the list of networks whose access must be corrected due to interference from TSPU equipment. Additions to the list are welcome. Please remember to cite the source.

## Guidelines
Simple guidelines (rules for the list):

The following are welcome:
Foreign hosting providers and CDN resources that suffer from interference from TSPU.

Do not add to the list:
YouTube and Google
Russian resources and networks.

## Legal notice
This list may be used by any telecom operators or users for any lawful purposes.
